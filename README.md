# Echo-Recording-Playback-system-for---Echo-dummy
Echo Vault — server-side sustav za Roblox igru. Svaka akcija igrača se snima pozicija/rotacija svakih 0.1s, a nakon 10 sekundi klonira se "eko" koji ponavlja istu putanju — ako aktiviraš zamku, eko će je aktivirati opet. Koristi Runservice.Heartbeat za snimanje i CFrame povjest za playback
